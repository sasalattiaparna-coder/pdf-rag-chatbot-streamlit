# Ask My PDF Bot

A PDF-based RAG Chatbot built using Streamlit, LangChain, FAISS, and OpenAI.

---

## Features

- Upload PDF files
- Ask questions from documents
- Education, Enterprise, and Legal domains
- RAG Architecture
- Fast similarity search using FAISS

---

## Technologies Used

- Python
- Streamlit
- LangChain
- OpenAI
- FAISS
- Sentence Transformers
